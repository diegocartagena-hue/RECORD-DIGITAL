# Sistema de Registro Digital - Escuela Interamericana

Sistema web para llevar el registro de anotaciones disciplinarias de estudiantes.

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Iniciar el servidor:
```bash
npm start
```

El servidor estará disponible en: **http://localhost:5000**

## Usuarios por Defecto

- **Administrador:** `admin@interamericana.edu` / `admin123`
- **Coordinador:** `coord@interamericana.edu` / `coord123`
- **Maestro:** `maestro@interamericana.edu` / `maestro123`

## Características

- Sistema de autenticación por roles
- Gestión de grados y estudiantes
- Sistema de anotaciones disciplinarias
- Dashboard con estadísticas y gráficas
- Notificaciones en tiempo real
- Botón de emergencia para maestros
- Importación masiva de estudiantes (Excel)
- Diseño responsive con efectos glass
- Tema claro/oscuro
