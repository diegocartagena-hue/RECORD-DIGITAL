============================================================
  QUE NECESITAS PARA ESTE PROYECTO
============================================================

NO NECESITAS:
--------------
- XAMPP (eso es para PHP/MySQL)
- Apache
- MySQL
- PHP

SOLO NECESITAS:
---------------
- Node.js (servidor de JavaScript)
- npm (viene con Node.js)

============================================================
  DIFERENCIA ENTRE XAMPP Y NODE.JS
============================================================

XAMPP:
- Para proyectos PHP
- Incluye Apache, MySQL, PHP
- NO lo necesitas para este proyecto

NODE.JS:
- Para proyectos JavaScript
- Es un servidor de JavaScript
- ES LO QUE NECESITAS para este proyecto

============================================================
  COMO INSTALAR NODE.JS
============================================================

1. Ve a: https://nodejs.org/
2. Descarga la version LTS (boton verde grande)
3. Ejecuta el instalador
4. IMPORTANTE: Marca "Add to PATH"
5. Completa la instalacion
6. Cierra todas las ventanas de CMD
7. Ejecuta: INICIAR.bat

============================================================
  DESPUES DE INSTALAR NODE.JS
============================================================

1. Haz doble clic en: INICIAR.bat
2. Espera a que instale las dependencias (primera vez)
3. El servidor iniciara automaticamente
4. Abre tu navegador en: http://localhost:5000
